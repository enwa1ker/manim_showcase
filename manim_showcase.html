Что такое Manim

Математика,
рассказанная визуально

Manim (Mathematical Animation Engine) — это библиотека Python, разработанная Грантом Сандерсоном (3Blue1Brown) для создания видеороликов по математике. Сообщество разработчиков создало собственный форк — Manim Community — с лучшей документацией и поддержкой.

С помощью Manim можно описать любой математический объект — от простого кружка до сложного преобразования Фурье — и анимировать его с точностью до пикселя.

Каждая сцена — это Python-класс. Каждый объект — Mobject. Каждое движение — Animation. Вся логика — в методе construct().

💡 Для научной работы: Manim идеально подходит для создания иллюстраций к теоремам, визуализации алгоритмов, объяснения геометрических доказательств и анимации статистических концепций.
🎬

Векторная точность

Все объекты рендерятся как векторная графика — никаких пикселей при масштабировании.

🐍

Чистый Python

Никаких GUI, никаких таймлайнов. Только код. Легко версионировать и воспроизводить.

🎨

LaTeX из коробки

Формулы рендерятся через LaTeX — красиво, точно, профессионально.

📐

Координатные системы

Оси, сетки, графики функций, трёхмерные сцены — всё встроено.

Установка

Начать за 3 шага

01

Зависимости

Установите Python 3.8+ и необходимые системные пакеты:

# macOS (Homebrew)
brew install ffmpeg

# Ubuntu/Debian
sudo apt install ffmpeg
sudo apt install texlive-full
02

Установка Manim

Установите библиотеку через pip:

# Создайте виртуальное окружение
python -m venv manim-env
source manim-env/bin/activate

# Установите Manim
pip install manim
03

Первый запуск

Создайте файл и запустите рендер:

# Низкое качество (быстро)
manim -ql scene.py MyScene

# Высокое качество (1080p)
manim -qh scene.py MyScene

# Превью в реальном времени
manim -p scene.py MyScene

АНИМАЦИЯ

Фигуры и трансформации

Базовые геометрические объекты — Circle, Square, Triangle — анимируются через метод play(). Можно рисовать контур, заливать цветом, трансформировать одну фигуру в другую.

Geometry Create / Write Transform
from manim import * class ShapesScene(Scene): def construct(self): # Создаём фигуры circle = Circle( radius=1.5, color=BLUE ) square = Square( side_length=2, color=YELLOW ) # Размещаем рядом circle.shift(LEFT * 2) square.shift(RIGHT * 2) # Рисуем контур self.play(Create(circle)) self.play(Create(square)) self.wait(1) # Трансформируем круг в квадрат self.play( Transform(circle, square), run_time=2 ) self.wait(1)

Графики функций

Axes создаёт систему координат. Метод plot() строит любую функцию Python. Можно анимировать построение, добавлять метки, подсвечивать области под графиком.

Axes plot() MathTex
from manim import * import numpy as np class FunctionGraph(Scene): def construct(self): # Создаём оси axes = Axes( x_range=[-4, 4, 1], y_range=[-2, 2, 1], axis_config={"color": WHITE}, ) # Строим функцию sin(x) graph = axes.plot( lambda x: np.sin(x), color=GREEN, x_range=[-4, 4] ) # Подпись формулы через LaTeX label = axes.get_graph_label( graph, label=MathTex(r"\sin(x)") ) # Анимация построения self.play( Create(axes), run_time=1 ) self.play( Create(graph), run_time=2 ) self.play( Write(label) ) self.wait()

Линейные преобразования

NumberPlane и LinearTransformationScene позволяют визуализировать действие матриц на пространство — как векторы сжимаются, растягиваются и вращаются. Отлично для лекций по линейной алгебре.

NumberPlane Vector ApplyMatrix
from manim import * import numpy as np class LinearTransform(LinearTransformationScene): CONFIG = { "include_background_plane": True, "show_coordinates": True, } def construct(self): # Матрица трансформации matrix = np.array([ [1, -1], [1, 1] ]) # Рисуем базисные векторы v1 = self.add_vector( [1, 0], color=GREEN ) v2 = self.add_vector( [0, 1], color=RED ) self.wait() # Применяем матричное преобразование self.apply_matrix(matrix) self.wait() # Показываем матрицу m = Matrix([[1, -1],[1, 1]]) self.play(Write(m)) self.wait(2)

Формулы LaTeX и доказательства

MathTex и Tex рендерят любые LaTeX-формулы. Можно анимировать появление формул посимвольно, подсвечивать части выражений, показывать алгебраические шаги.

MathTex Write TransformMatchingTex
from manim import * class PythagorasProof(Scene): def construct(self): # Заголовок title = Tex( "Теорема Пифагора", font_size=36 ) title.to_edge(UP) self.play(Write(title)) # Формула formula = MathTex( r"a^2 + b^2 = c^2", font_size=56 ) self.play(Write(formula)) self.wait() # Подсвечиваем части self.play( formula[0][0:2].animate.set_color(BLUE), formula[0][3:5].animate.set_color(GREEN), formula[0][6:8].animate.set_color(YELLOW), ) # Треугольник triangle = Polygon( [-2, -2, 0], [2, -2, 0], [2, 1, 0], color=WHITE ) self.play( formula.animate.to_edge(UP), Create(triangle) ) self.wait(2)
Ключевые концепции

Архитектура Manim

📦

Mobject

Mathematical Object — базовый класс всех объектов сцены. Круги, тексты, стрелки — всё Mobject.

🎬

Animation

Описывает, как изменяется Mobject во времени. Create, Write, Transform, FadeIn...

🎭

Scene

Контейнер для всей сцены. Метод construct() — место, где описывается анимация.

⏱️

self.play()

Запускает одну или несколько анимаций одновременно. run_time контролирует скорость.

📐

ValueTracker

Хранит число, которое можно анимировать. Связывает параметры объектов через always_redraw.

📝

MathTex / Tex

Рендерит LaTeX-формулы. MathTex для математики, Tex для обычного текста с формулами.

📊

Axes / NumberPlane

Координатные системы для построения графиков и визуализации преобразований.

🔄

Updater

Функция, вызываемая каждый кадр. Позволяет создавать зависимые, реактивные объекты.

Структура типичного скрипта:

from manim import *   # импортируем всё

class МояСцена(Scene):
    def construct(self):
        # создаём объекты...
        self.play(...)  # анимируем
        self.wait(1)   # пауза
